## ⚙️ Prerequisites

- python 3.14.2
- Django 6.0.1

## 🚀 Installation

```bash
git clone https://github.com/shm233/Authentication-Basics
cd Authentication-Basics
```

```
create virtual environment in cmd(command prompt)
python -m venv env
.env\Scripts\activate
cd newproject
```


## 💻 Usage

```bash
python manage.py runserver
```
